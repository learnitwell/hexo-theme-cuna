## Installation

### Install

``` bash
$ git clone https://github.com/learnitwell/hexo-theme-cuna.git themes/cuna
```

**cuna requires Hexo 2.4 and above.**

### Enable

Modify `theme` setting in `_config.yml` to `cuna`.

### Update

``` bash
cd themes/cuna
git pull
```

## Configuration

``` yml
# Header
menu:
  Home: /
  Archives: /archives
  About: /about
rss: /atom.xml

```
